## ğŸŒŸ Fitur Utama

Sesuai dengan spesifikasi tugas akhir modul, kalkulator ini memiliki fitur-fitur berikut:

### 1. Fungsionalitas Dasar
* **Operasi Aritmatika:** Mendukung penambahan (+), pengurangan (-), perkalian (Ã—), dan pembagian (Ã·).
* **Perhitungan Berantai:** Dapat menghitung ekspresi matematika panjang sekaligus dengan urutan operasi yang benar (contoh: `5 + 3 Ã— 2` akan menghasilkan `11`, bukan `16`).
* **Desimal:** Mendukung angka desimal (titik).
* **Reset:** Tombol `C` (Clear All) dan `CE` (Clear Entry/Backspace).

### 2. Fitur Lanjutan (Advanced)
* **Riwayat Perhitungan (History):** Menyimpan dan menampilkan 5 perhitungan terakhir secara dinamis di samping kalkulator.
* **Fungsi Memori:**
    * `M+`: Menambah angka di layar ke memori.
    * `M-`: Mengurangi angka di layar dari memori.
    * `MR`: Memanggil kembali angka dari memori (Memory Recall).
    * `MC`: Menghapus isi memori (Memory Clear).
* **Penanganan Error:** Menampilkan pesan peringatan (Alert) jika terjadi pembagian dengan nol atau error sintaks.

### 3. Antarmuka & UX
* **Desain Responsif:** Menggunakan **Bootstrap 5**, tampilan menyesuaikan dengan baik di perangkat Mobile maupun Desktop.
* **Keyboard Support:** Kalkulator dapat dioperasikan menggunakan keyboard fisik (Numpad) untuk pengalaman penggunaan yang lebih cepat.
* **Styling Kustom:** Desain visual meniru gaya kalkulator fisik klasik (Casio Style) dengan layar LCD hijau dan tombol timbul.
